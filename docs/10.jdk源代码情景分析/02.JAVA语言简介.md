### 1.1、一些过往的历史

Java® 编程语言是一门通用的、并发的、面向对象的编程语言。它的结构类似于C和C++, 但它省略了许多使C和c++复杂、混乱和不安全的特性。

Java平台最初是为了解决为联网消费设备构建软件的问题而开发的。它的设计目的是支持多主机体系结构，并允许安全交付软件组件。为了满足这些需求，编译后的代码必须能够跨网络传输，在任何客户端上运行，并确保客户端能够安全运行。
